# Big Train

## A training and coaching community

Big Train (the working name) is an application to help athletes and coaches to:

* connect with each other
* manage their training
* provide feedback

### Development stack

#### Server

* Node.js
* Express.js
* Jade
* Coffeescript
* Stylus

#### Client

* RequireJS
* Backbone
* Underscore
* Handlebars
* Qwery
* Ender.js
* Coffeescript
